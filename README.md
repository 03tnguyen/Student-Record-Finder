## About The Project

This project is a student record management system that allows users to view, add, remove, and back up student data stored in a binary file, providing a menu-driven interface to perform these operations.

## Getting Started

### Executing program
* grant permission
  ```sh
  chmod u+x build.sh
  ```
* run program
  ```sh
  ./build.sh
  ```
